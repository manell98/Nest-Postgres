## Description

Projeto desenvolvido em NestJS e utilizando o banco de dados Postgres apenas para fins de estudo

## Installation

```bash
$ npm install
```

## Running the app

```bash
# postgres
$ docker-compose up

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm test
```
